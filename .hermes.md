# Markov Regime Analyst — Hermes Context

## What This Is
Kareem's market-regime analysis app. Detects market state (Bull / Sideways /
Bear) using a Markov model and visualizes it. Personal tool + portfolio demo.
Shares the regime concept with Word Money System and Stock Hawk.

## Stack
Next.js (App Router, TypeScript), Tailwind. Dockerfile present for container
deploy. Scripts in `scripts/`.

## What It Does
Markov regime detection over price series. v2 added a price chart with a
regime ribbon overlay (states colored along the timeline) plus transition
probabilities between states.

## Status
v2 COMPLETE as of May 2026. Working app. Also deployed as a standalone
portfolio demo.
